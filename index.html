About Me

從多媒體設計領域跨入程式世界

我是游玟馨，從熱情成為實踐者。

為什麼要踏進程式世界？大學畢業後進入企業，發現原有的公司網站已不具有現今網站的需求，在職時和網頁設計公司討論新網站的建置，發現相較於靜態的網頁資訊，動態的網頁更能給使用者互動的感覺。

在進入網頁開發領域後，開始上網購買課程，下班後線上學習，對於網頁設計越發熱情，辭職後進入職訓參訓半年，從零開始，從學會程式語言基本語法到響應式網頁切版再到框架應用，最終能獨立建立完整的前後端整合網站。

我喜歡解決問題後的成就感，從多媒體設計領域進入網頁設計，接著網頁程式開發。學習過程中，會有許多從沒遇過的問題需要被解決，解決問題時，學會使用網路及社群的力量、詢問職訓的老師，在Debug的過程裡，從錯誤中學習。目前對於網頁UI/UX，到資料庫的串接，前後端相互配合，都有了經驗。

涉獵到程式的領域中，能持續學習、精進技術是關鍵，期許將來能夠開發出更優良的作品，為公司帶來價值。

美好自遊

美國自由行攻略平台網站，透過地圖和閱覽景點評價功能，讓使用者旅遊時不失去方向感，也能一覽附近美食、景點，此平台有票券商城讓使用者能即刻訂票。

View demo

『美好自遊』概念發想

多次到美國遊玩的經驗，發現安排多天的行程會遇到以下問題:

  • 1. 安排行程還需另尋購票渠道的問題
  • 2. 查看景點時需到其他網站搜尋評論、評分
  • 3. 行前注意事項需查閱各類網站文章

此網站希望可以解決以上問題，將旅客從旅行前後會遇到的問題在此平台中都可解決。

使用者前台

前台功能:

  • 1. 會員註冊/登入
  • 2. Google Map地點自動搜尋
  • 3. TravelAdvisor景點評論網站結合地圖
  • 4. 加入、修改購物車
  • 5. 結帳、送出訂單
  • 6. 行前攻略文章篩選查看、收藏
  • 7. 票券篩選查看、收藏
  • 8. 查看已結帳訂單、票券及文章收藏
  • 9. 會員行事曆功能

前台使用技術:

  • 1. 使用 Quasar 框架開發
  • 2. 以Pinia存取使用者資訊、API回傳資料
  • 3. Vue Router 頁面切換
  • 4. SCSS 預處理器
  • 5. Google Map Platform API
  • 6. axios 套件處理 AJAX
  • 7. validator 信箱驗證

管理者後台

後台功能:

  • 1. 新增、修改及管理票券、文章
  • 2. 查看會員資料
  • 3. 管理訂單

前後端整合:

  • 1. 使用 Node.js + Express 框架
  • 2. RESTful API 進行請求
  • 3. CORS 處理跨域請求
  • 4. Passport 處理身份驗證
  • 5. bcrypt 處理密碼加鹽
  • 6. MongoDB + Mongoose 套件，建立 Schema
  • 7. Multer 套件處理上傳文件
  • 8. Cloudinary 進行媒體管理
